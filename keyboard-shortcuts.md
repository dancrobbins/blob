# Keyboard shortcuts

These shortcuts apply when the caret is in an active text insertion point (editing a blob).

| Action | Windows | Mac |
|--------|---------|-----|
| Move current bullet point **before** the preceding bullet point (if there is one) | **Ctrl+Alt+↑** | **Ctrl+Option+↑** |
| Move current bullet point **after** the next bullet point (if there is one) | **Ctrl+Alt+↓** | **Ctrl+Option+↓** |
| Indent the current bullet point by one level | **Tab** | **Tab** |
| Unindent the current bullet point by one level (no effect if already at indent 0) | **Shift+Tab** | **Shift+Tab** |
